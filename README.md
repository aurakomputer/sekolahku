## Fitur / Module
- CMS (Content Management System)
  - Portal Berita
  - Kegiatan Sekolah
  - Informasi Sekolah
  - Daftar Guru
  - Gallery
  - Profl Alumni
- Database (Buku besar siswa)
  - Management data pribadi siswa
  - Riwayat kelas siswa
- Pendaftaran Siswa Baru
  - Form pendaftaran
  - Seleksi penerimaan
  - Pembayaran Daftar Ulang
  - Pembagian Seragam
  - Pembagian Kelas siswa baru
